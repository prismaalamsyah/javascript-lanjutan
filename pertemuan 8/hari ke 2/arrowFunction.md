# Penggunaan Arrow Function
## Function Expression
```js
let tampilPesan = function (nama) {
    alert('halo' + nama);
}
tampilPesan('Prisma Alamsyah');
```
## Arrow Function ?
> _Bentuk lain yang lebih ringkas dari Function Expression_
**- MDN**
```js
let tampilPesan = (nama) => {
    alert('halo' + nama);
}
tampilPesan('Prisma Alamsyah');
```
